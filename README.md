# Python协同过滤电影推荐系统 python165

## 项目概述

本项目是一个基于Python的协同过滤电影推荐系统，包括源代码和详尽的文档（约2万字）。该系统采用Django框架，结合SQLite数据库，以及HTML、CSS、JavaScript等前端技术构建。

## 技术栈

- **后端：** Django
- **数据库：** SQLite
- **前端：** HTML, CSS, JavaScript

## 功能模块

系统主要包含以下功能模块：

- **电影信息：** 展示电影详情
- **电影资讯：** 提供电影相关的新闻和资讯
- **电影搜索：** 支持电影搜索功能
- **电影收藏：** 用户可以收藏喜欢的电影
- **电影评分：** 用户可以对电影进行评分
- **电影分类管理：** 管理电影的分类
- **用户管理：** 管理用户账户
- **电影评分管理：** 管理电影的评分数据
- **系统管理：** 对整个系统进行配置和管理

## 系统角色

- **普通用户：** 可以浏览电影、进行评分和收藏电影等
- **管理员：** 管理电影信息、用户和系统设置等
